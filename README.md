## Belangrijk: eenmalige Supabase tabel maken

Voor Teamchat, Mededelingen en Activiteitenlog moet je één nieuwe tabel maken in Supabase.

Ga naar Supabase → SQL Editor → New query en plak:

```sql
create table if not exists hub_items (
  id bigint generated by default as identity primary key,
  created_at timestamptz default now(),
  type text not null,
  author_id bigint,
  author_name text,
  message text,
  pinned boolean default false,
  target text default 'all',
  read_by jsonb default '[]'::jsonb,
  meta jsonb default '{}'::jsonb
);
```

Laat RLS voorlopig uitgeschakeld voor deze tabel, zoals bij de andere tabellen tijdens deze testfase.

## Vercel projectnaam en URL

De URL `project-qnhpq.vercel.app` verander je niet in de code.
Dat doe je in Vercel:
Project → Settings → General → Project Name = `tuinafsluiter-hub`

Voor een mooie URL:
Project → Settings → Domains → voeg later toe:
`hub.tuinafsluiter.be` of `prikklok.tuinafsluiter.be`
