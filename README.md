## Supabase: éénmalige tabel
Voor Teamchat, Mededelingen en Activiteitenlog:

```sql
create table if not exists hub_items (
  id bigint generated by default as identity primary key,
  created_at timestamptz default now(),
  type text not null,
  author_id bigint,
  author_name text,
  message text,
  pinned boolean default false,
  target text default 'all',
  read_by jsonb default '[]'::jsonb,
  meta jsonb default '{}'::jsonb
);
```

Laat RLS voorlopig uit tijdens de testfase, zoals bij de andere tabellen.

## iPhone
Na deploy:
1. Verwijder oude Tuinafsluiter Hub van beginscherm.
2. Open de site in Safari.
3. Deel → Zet op beginscherm.

Dan wordt het nieuwe icoon en de naam correct geladen.
